Exercícios - Arquivos
---------------------

1. Crie um scrit para CLI que imprima o caminho e o tamanho, em KB (KiloBytes), de todos os arquivos PHP no diretório atual

2. Modifique o script anterior para aceitar o padrão de busca (pattern) como argumento do script. Por exemplo: php listar.php *.txt - deve listar todos os arquivos de texto do diretório atual

3. Faça um script que leia um arquivo e imprima quantas vogais há nesse arquivo

4. Faça um script que leia um arquivo PHP e informe quantos comandos há nele.

5. Fala um script que grave, em um arquivo numeros.dar, 50 números gerados aleatoriamente, cada um em uma linha diferente.
